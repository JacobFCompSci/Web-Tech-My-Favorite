WEB TECHNOLOGIES

HTML

HTML (HyperText Markup Language) is the fundamental building block of the Web and define the content and structure of a web page. Below are some common page elements that HTML is used for:

  • Headings
  • Paragraphs
  • Images
  • Links
  • Lists

Learn more about HTML with the w3 schools HTML Turorial

CSS

CSS (Cascading Style Sheets) is used to describe a web page's appearance and presentation. Below are some common uses of CSS:

  • Fonts
  • Colors
  • Layout
  • Backgrounds

Learn more about CSS with the w3schools CSS Tutorial

JAVASCRIPT

Javascript is the programming language of the web and controls the functionality and behaviour of a web page. Below are some common uses of Javascript:

  • Add Interactivity to a Web Page
  • Create Web Apps
  • Create Web Games & Animations

A great JavaScript reference is The Modern Javascript Turorial
